# StudyLink — Allemand fonctionnel complet A1 à C1

Le parcours est installé depuis l'administration avec le bouton **🇩🇪 Allemand A1 → C1**.

## Contenu

- 5 niveaux : A1, A2, B1, B2, C1
- 12 modules par niveau
- 8 leçons par module
- 60 modules au total
- 480 leçons au total
- 60 quiz de validation
- 60 missions réelles
- 60 ateliers intensifs avec 12 exercices chacun
- environ 600 heures de parcours guidé et de pratique

## Fonctionnalités

Chaque leçon du cours possède :

- une explication en français ;
- des exemples en allemand ;
- des exercices d'application ;
- une activité orale ;
- un rituel de révision espacée ;
- un lecteur audio allemand basé sur la voix du navigateur ;
- un réglage de vitesse ;
- une fiche téléchargeable ;
- une impression / export PDF ;
- des ressources vidéo externes sur les leçons clés ;
- la progression et les notes personnelles StudyLink.

## Installation

1. Déployer le backend contenant `019_complete_german_a1_c1.sql`.
2. Ouvrir StudyLink avec un compte administrateur.
3. Aller dans **Administration → Cours**.
4. Cliquer sur **🇩🇪 Allemand A1 → C1**.
5. Attendre le message de succès puis ouvrir le catalogue.

Le script est idempotent : le relancer met à jour le cours sans créer de doublons de modules ou de leçons.
